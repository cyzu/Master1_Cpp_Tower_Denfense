LANCER LE PROGRAMME :
	1) Télécharger ce projet
	2) Extraire, puis l'ouvrir depuis xcode, puis lancer le programme.

COMMENT JOUER :
Au lancement du jeu, vous avez le choix entre lancer une vague où poser des vaisseaux.

	Chaque lancement de vague s'effectue avec la touche 'Entrée'. Si il y a une vague lancée, sans qu'elle soit terminée, la touche 'Entrée' ne créera pas de nouvelle vague. Il faut attendre la fin de celle-ci.
Après chaque fin de vague, un message sera affiché (en foncé, très discret) au milieu de la fenêtre.

	Pour poser un vaisseau, il suffit de cliquer sur une case au choix. En cliquant plusieurs fois sur la même case, vous pouvez faire défiler les différents vaisseaux. Lorsque vous avez fait votre choix, validez avec 'Entrée'.
	Il est possible de changer de case avant de poser un vaisseau : il suffit simplement de cliquer sur une autre case.
	Il est possible d'annuler la pose d'un vaisseau avec 'Delete'. Cependant, une fois le vaisseau posé, il n'est pas possible de l'enlever.
	Il est de même pas possible de poser plusieurs vaisseaux sur une même case.

	Il est possible de mettre pause une partie avec la touche 'Espace'. Lorsque la pause est activée, aucun vaisseau ne peut être posé.

	A la fin de la partie, faites 'Entrée' sur fermer le jeu

	A tout moment vous pouvez changer la taille de la fenêtre. Cela n'impactera pas le graphisme des éléments du jeu.
